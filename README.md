# dash-finan
📊 Modern Finance Dashboard Um dashboard de finanças pessoais offline-first, com interface minimalista e foco em alta performance visual. Este projeto oferece uma solução completa para controle de rendas e gastos mensais, garantindo total privacidade e uma análise clara de dados financeiros através de gráficos dinâmicos.

✨ Funcionalidades
Fluxo de Caixa Flexível: Gestão de múltiplas entradas de renda (salário, rendas extras, bônus) com histórico detalhado.

Categorização Avançada: Gastos organizados por categorias customizáveis com feedback visual imediato.

Interatividade Otimizada: Navegação fluida via teclado (atalhos "Enter" para salvar e pular campos) e formatação monetária nativa.

Análise Visual de Dados: Dashboard estatístico integrado com gráficos de Rosca e Barras (via Chart.js) para visualização de saúde financeira.

Arquitetura Offline: Persistência de dados local via localStorage. Seus dados nunca saem do seu dispositivo, garantindo 100% de privacidade.

Portabilidade de Dados: Sistema de exportação e importação em formato .json para backups seguros.

Multi-período: Navegador de meses independente para acompanhamento histórico de longo prazo.

Design Contemporâneo: Interface baseada em princípios de Clean Design, com sombras suaves, bordas orgânicas e tipografia otimizada.

🚀 Tecnologias Utilizadas
HTML5 & CSS3: Layout estruturado com Flexbox e CSS Grid.

JavaScript (ES6+): Lógica funcional para manipulação de estados e persistência.

Chart.js: Biblioteca para visualização de dados dinâmica.

Google Fonts: Tipografia focada em legibilidade (Inter).

📦 Instalação e Uso
O projeto é estático e independente, não necessitando de backend ou instalação de dependências:

Clone o repositório ou baixe o arquivo .html.

Abra o arquivo diretamente em seu navegador.

Seus dados serão salvos automaticamente conforme você preenche os campos.
